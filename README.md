# Projet Node - Tournoi de Tarot

Pour lancer le projet:

- Clonez le projet avec [https://github.com/ormartins/Node.git](https://github.com/ormartins/Node.git)
- Rendez-vous à la racine du projet et lancez la commande npm install pour installer les dépendances
- Lancer le script db\_tournoi.sql dans votre SGBD
- Lancez ensuite le serveur avec npm start et rendez-vous à l&#39;adresse localhost:5000 pour naviguer l&#39;application
